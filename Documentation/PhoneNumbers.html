Important Note about phone numbers

Basic Concepts

We use the following plugin for inputting phone numbers:

https://github.com/Bluefieldscom/intl-tel-input

http://jackocnr.com/intl-tel-input.html

We also use google phone number validation script to validate phone number.

Details are provided here:

https://code.google.com/p/libphonenumber/

We store phone number in 'telefono' column in Usuarios table


How to store phone numbers in this column ?

eg : Your phone number is +1 469 579 7083

  • Step 1 : Remove all spaces,hiphens,etc --> +14695797083
  • Step 2 : Remove + sign --> 14695797083
Store this in database. Make sure there are no garbage(invisible) characters at end of the phone number


How to display phone numbers in this plugin ?

  • Retrieve phone number from database
  • Add '+' sign in front of the phone number (Always do this)
  • Provide it to the plugin
