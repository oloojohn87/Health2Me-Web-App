Telemedicine Phone

Basic Overview


->Telemedicine phone feature is provided by Health2Me for members accounts which allows them to do phone consultation with the doctors of Health2Me network. The feature is provided through two buttons on member app, Search and Talk.

->Phone consultation can be started immediately based on doctor avaliability or it can be scheduled for later time through setting up appointment through appointment button in Talk button or "Schedule" button as seen below in the figures.

If an appointment is set for a later time, actually, the member selects a time range while setting up the appointment. The doctor selects an appropriate time based on his/her availability. Both the doctor and memeber receives a call at that time. Back end process keeps on checking appointments table and initiates call if there are any appointments.


->Prior to starting the phone consultation, credit card has to be set up in the card details section of "Set Up" button.

-> Phone consultations are chargeable to the members based on the tier they are set up.

Working


->The technology used behind for telemedicine phone consultation is Twilio APIs'. We use Twilio at the back end to initiate the phone consultation. Important source files which are associated with the ameridoc_call_doctor.php,ameridoc_connect.php,phone_appointment_ask.php,phone_appointment_callback.php,phone_appointment_handle_leave, phone_appointment_wait and start_telemed_phone.php
