Functions Name Outside of Classes:

Lowercase with words underscore delimited Abbreviations should be used sparingly so that code is readable. Try to minimize letter count. Try to minimize use of abbreviations. Good: 'mcrypt_enc_self_test' 'mysql_list_fields' Ok: 'mcrypt_module_get_algo_supported_key_sizes' (could be 'mcrypt_mod_get_algo_sup_key_sizes'?) 'get_html_translation_table' (could be 'html_get_trans_table'?) Bad: 'hw_GetObjectByQueryCollObj' 'pg_setclientencoding' 'jf_n_s_i'

Methods Name Inside of Classes:

Methods can be camelCase. The initial word should be lower case and the words following the first should be upper case. Good: 'connect()' 'getData()' 'buildSomeWidget()' Bad: 'get_Data()' 'buildsomewidget' 'getI()' Variable Names: Variable names should be meaningful. Variables should be lowercase and underscore delimited.
