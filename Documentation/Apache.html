Apache Startup Procedure

Health2me uses Windows machines and Apache Web Server. There are three basic environments:

  • Development Server
  • Beta Server
  • Production Server

Development and Beta servers use XAMPP for Apache, MySQL and Filezilla. Production server uses XAMPP for Apache and MySQL, but APache runs separately.

Apache server 2.4 Startup Procedure for Production Server

Apache server 2.4

A new apache server 2.4 has been installed as part of the migration at location C:\Apache24. Therefore the main directory for the health2me application starts at “C:\Apache24\htdocs”

Below are the set of configuration file and location for apache server for changing settings:

  • httpd ­ C:\Apache24\conf
  • SSL configuration ­ ssl.crl,ssl.crt,ssl.csr,ssl.key

Starting server

To start the server open the command prompt and enter httpd ­k start

Stoping server

To stop the server open the command prompt and enter httpd ­k stop

PHP version

Latest version of PHP 5.4.17 has been installed to utilise the best available features of the language.

The main directory C:\PHP is the place where it is stored.

For changing any configuration related to PHP, we need to modify php.ini file which resides in C:\php\php.ini
