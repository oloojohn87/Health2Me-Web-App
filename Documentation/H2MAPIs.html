Health2Me .- APIs

ICD-10 API

This API performs an ICD-10 code look up.

Used in

  • medicalPassport.php (version for Doctors only)

License

  • World Health Organization commercial distribution license

Instructions

Name: xmlread.php

Input: http://beta.health2.me/xmlread.php?search_string=influen

Output: JSON object with the name of the matched diseases and their ICD-10 codes:
