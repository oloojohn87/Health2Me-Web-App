Processes

1.- Appointment Manager (node appointment_manager.js)

The Appointment Manager is a program written in Javascript that is run through node.js (using Google's V8 Javascript engine). The purpose of the Appointment Manager is to constantly check all of the telemedicine appointments that are currently in the system (in the appointments data table) and initiate them when appropriate. The program is located in the root directory with the file name appointment_manager.js, and it can be run as a process using the command node appointment_manager.js.

The Appointment Manager is dependent on the Mysql and Twilio node modules. They should already be installed on the server. If they are not, they can be installed using the commands npm install mysql and npm install twilio respectively in the same directory as the appointment_manager.js file.
