Timeline



The timeline is a H2M specific plugin that displays recent activity information between a given doctor and patient.


Dependencies

The timeline is dependent on a few js and css files and its backend file to generate content.

Javacript:

  • js/jquery.min.js
  • js/jquery.easing.1.3.js
  • js/jquery-ui.min.js
  • js/tipped.js
  • js/imagesloaded.pkgd.min.js
  • js/H2M_Timeline.js

CSS:

  • css/tipped.css
  • font-awesome/css/font-awesome.min.css
  • css/jquery-ui-1.8.16.custom.css

PHP:

  • get_personal_doctor_timeline.php

Getting Started

To create a timeline, just simply add a div to an html page, and then in the $(document).ready function of JQuery call the H2M_Timeline function to your div. Make sure the div is at least 25 pixels in height.

HTML:

    <div id="main_timeline"></div>

JQuery:

    $("#main_timeline").H2M_Timeline({doctor: doc_id, patient: pat_id, max: 30});


Parameters

You must pass three parameters to the H2M_Timeline function for it to display the content properly.

Parameters:

doctor - The id of the doctor that will be used to display this timeline

patient - The id of the patient that will be used to display this timeline

max - The maximum number of items to return
